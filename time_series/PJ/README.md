# Telegram-бот для прогнозирования акций (Итоговый проект)

## Что делает бот
Телеграм-бот, который:
1) Принимает тикер акции (например, AAPL) и сумму «условной инвестиции».
2) Загружает дневные цены за последние 2 года с Yahoo Finance (через `yfinance`).
3) Обучает 3 разных подхода к прогнозированию:
   - ML-модель с лаговыми признаками (Ridge Regression)
   - Статистическая модель (ARIMA с подбором параметров по сетке)
   - Нейросетевая модель (LSTM)
4) Оценивает модели на хронологическом разбиении train/test и выбирает лучшую по RMSE (при равенстве — по MAPE).
5) Строит прогноз на ближайшие 30 торговых дней и отображает график «история + прогноз».
6) Формирует рекомендации покупки/продажи по локальным минимумам/максимумам прогнозного ряда.
7) Оценивает гипотетическую прибыль для указанной суммы (только в учебных целях).
8) Логирует каждый запрос в `logs/logs.csv`.

## Быстрый старт (Docker)
1) Укажите токен Telegram-бота в `.env`:
   ```bash
   cp .env.example .env
   ```
2) Запустите:
   ```bash
   docker compose up -d
   ```
3) Откройте бота в Telegram и отправьте:
   ```
   AAPL 1000
   ```
   или
   ```
   /help
   ```
